# Repositorio colaborativo

1 - git clone https://github.com/AntonioAlberto25/Projeto-Confeitaria-FastAPI---Python.git

2 - git add . -> Adicionar arquivos no git para versionamento

3 - git commit -m "TEXTO" -> Adicionar commit local

4 - git push origin main -> Enviar arquivos para GitHub

5 - git pull origin main -> Baixar commits do GitHub

6 - Criar branch secundária = git checkout -b "feature/minha-tarefa"

7 - Subir branch = git push -u origin "feature/minha-tarefa"

8 - Para atualizar: git checkout main
                    git pull origin main
